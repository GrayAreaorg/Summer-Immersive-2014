Adding Scripts to HTML

The
Document Object Model

The DOM is the interface that allows us to
manipulate HTML documents.

Most of the DOM is accessible via the
global variable document.

Nodes

In the DOM, elements are referred to as nodes.

Finding Elements

Here, node can be either document or a specific node.

  • document.getElementById(id)
  • node.getElementsByTagName(name)
  • node.getElementsByClassName(class)
  • node.querySelector(selector)
  • node.querySelectorAll(selector)

DOM Element Properties

  • node.parentNode is the node’s parent
  • node.firstChild is the node’s first child
  • node.lastChild is its last child
  • node.childNodes is a list of the node’s children:
    • node.childNodes.length tells you how many children it has
    • node.childNodes[index] or
    • node.childNodes.get(index) returns an indexed child node

Modifying Elements

  • assigning node.innerText modifies text (not HTML)
  • assigning node.innerHTML rewrites the node’s children
  • node.setAttribute(attr, value)
  • node.removeAttribute(attr)
